# Hi, I'm Kevin Díaz Madrigal 👋

QA Engineer / SDET with 9+ years of experience turning manual QA bottlenecks into automated, reliable testing workflows across web, mobile, API, accessibility and email.

## What I'm working on

At [VML](https://www.vml.com), I build test automation for enterprise automotive, telecom and marketing platforms — including a Playwright + TypeScript UAT framework for Honda and Acura model launches, and QA integrations connecting Jira, Zephyr, Azure DevOps, Figma and Microsoft Graph into testing workflows. Lately I've been focused on AI-assisted QA: MCP servers, GitHub Copilot, OpenCode and Claude Code applied to day-to-day testing tasks.
